# My 3D Printing Notes

这是我的 3D 打印学习笔记，主要记录 Bambu P2S、PLA 打印、Fusion 360 建模、Bambu Studio 切片，以及 GitHub 上找到的相关资源。

## 学习目标

- 学会使用 GitHub 管理学习资料
- 学会收集和筛选开源项目
- 学会记录 3D 打印参数和失败经验
- 完成一个自定义尺寸的沥水篮或收纳盒

## 当前设备和软件

- 3D 打印机：Bambu Lab P2S
- 材料：PLA
- 建模软件：Fusion 360
- 切片软件：Bambu Studio
- 版本管理：GitHub
